# Team Images

Place team member photos here.

## Image Guidelines

- **Format**: JPG or WebP
- **Aspect Ratio**: 3:4 portrait (e.g., 900x1200px)
- **Style**: Professional portraits, preferably with neutral background
- **Size**: Optimize to under 500KB per image
- **Naming**: Use role-based names (e.g., `guide-1.jpg`, `coordinator-1.jpg`)

## Updating Team Info

Edit the `teamMembers` array in `/components/about/TeamSection.tsx`:

```typescript
{
  name: 'Full Name',
  role: 'Job Title',
  experience: 'X years',
  languages: ['English', 'Urdu', 'Arabic', 'Telugu'],
  image: '/images/team/filename.jpg',
  specialization: 'Area of expertise'
}
```
